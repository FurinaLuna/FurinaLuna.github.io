# 职场礼仪与沟通技巧学习笔记

## 一、禁止的目光
在社交与职场沟通中，以下几种目光是需要避免的：
1. 阶级斗争眼
2. 无所畏惧眼
3. 四大皆空眼
4. 虎口脱险眼
5. 高高在上眼
6. 低眉信首眼
7. 上下打量眼
8. 蔑视眼等

同时，不同的视线方向传递不同的情绪：
- 视线向下：表现权威感和优越感
- 视线向上：表现服从与任人摆布
- 视线水平：表现客观和理智


## 二、礼仪与沟通的自测清单
可以通过以下清单自我检查礼仪与沟通是否到位：
- [ ] 着装得体，符合场合要求
- [ ] 主动问候，使用礼貌用语
- [ ] 工作区域整洁有序
- [ ] 电话邮件及时回复
- [ ] 倾听他人，不随意打断
- [ ] 表达清晰，重点突出
- [ ] 情绪稳定，态度积极
- [ ] 任务跟进，及时反馈


## 三、五说五不说
在沟通表达中，要把握“五说五不说”的原则：
| 不说       | 说       |
|------------|----------|
| 原因       | 情况     |
| 责任       | 措施     |
| 教训       | 决定     |
| 损失       | 成果     |
| 性质       | 希望     |

**注意**：没有结论、没有决定的事情不说；有定论、不会更改的内容可以说。
